# Agent A / Summarizer 第一轮独立总结模板

## 固定输入

- 书籍目录：`{BOOK_DIR}`
- 主书稿：`{BOOK_MD}`
- 章节目录：`{CHAPTERS_DIR}`
- 图片清单：`{IMAGE_MANIFEST}`
- 输出目标：`{OUTPUT_TARGET}`
- 体裁与子体裁：`{GENRE}`
- 用户额外要求：`{USER_REQUIREMENTS}`
- Agent A 输出：`{AGENT_A_OUTPUT}`
- Agent B 输出：`{AGENT_B_OUTPUT}`

本阶段不得读取 `source/` 中的原始书籍文件。若 `{CHAPTERS_DIR}` 或 `{IMAGE_MANIFEST}` 不存在，基于 `{BOOK_MD}` 继续，但必须说明该输入缺失。

## 任务

你是 Agent A / Summarizer。请独立完整阅读 `{BOOK_MD}`，必要时参考 `{CHAPTERS_DIR}`，生成忠实原文的深度总结初稿材料。第一轮完成前不得接触 Agent B 的任何输出。

## 输出要求

请按以下三个部分输出，不要写最终正式总结文件：

1. `summary_draft`
   - 先识别体裁与子体裁。
   - 按原书目录、章节和小标题逐级总结。
   - 非虚构侧重中心论点、论证链条、证据、概念框架与局限。
   - 小说/文学侧重情节结构、人物弧光、主题、叙事技巧、意象与情感基调。
   - 历史/传记/回忆录侧重历史脉络、关键人物、因果关系、作者视角与偏向。
   - 明确区分“原书内容”“总结归纳”和“解释性判断”。
   - 除非用户额外要求中明确要求批判性评价，不得生成对原书的批判性评价；若用户明确要求，必须单独标注。

2. `coverage_ledger`
   - 列出已覆盖的章节、小节或明显结构单元。
   - 标出可能缺失、模糊、转换异常或需要主线程复核的位置。

3. `argument_map`
   - 对非虚构、学术、商业/自助、哲学类书籍，绘制核心论点、分论点、证据和局限。
   - 对小说、文学、历史、传记、回忆录类书籍，绘制叙事结构、人物/事件因果链和主题线索。

## 质量边界

- 不得虚构情节、观点、引文、页码或作者意图。
- 没有原文证据的判断必须标为解释性判断或待复核。
- 章节覆盖优先于压缩表达，事实忠实度优先于文采。
